# CompanyIntel 🧠

AI-Powered Lead Intelligence & Automation Platform

## 📋 Overview

CompanyIntel is an intelligent lead automation system that automatically enriches prospect data using AI, generates professional audit reports, and sends them via email. Built with React, Express, and Groq AI.

### Key Features

- ✅ **Automated Lead Processing** - Captures and processes lead submissions instantly
- ✅ **AI-Powered Enrichment** - Uses Groq Mixtral AI for intelligent company analysis
- ✅ **Professional PDF Reports** - Generates beautifully formatted audit reports
- ✅ **Automated Email Delivery** - Sends reports directly to prospects
- ✅ **Error Handling** - Intelligent fallback system ensures reliability
- ✅ **Real-time Processing** - Complete workflow in seconds

## 📦 Installation & Setup

### Prerequisites
- Node.js (v14+)
- npm
- Gmail account with App Password
- Groq API key

### Step 1: Clone Repository
```bash
git clone https://github.com/yourusername/CompanyIntel.git
cd companyintel
```

### Step 2: Setup Backend

```bash
cd backend
npm install
```

Create `.env` file:

### Step 3: Setup Frontend

```bash
cd frontend
npm install
```

### Step 4: Get API Keys

**Groq API Key:**
1. Go to https://console.groq.com/keys
2. Create new API key
3. Add to backend/.env

**Gmail App Password:**
1. Go to https://myaccount.google.com/apppasswords
2. Select Mail & Windows Computer
3. Copy 16-char password
4. Add to backend/.env

## 📝 How to Use

1. **Open Form** - Go to http://localhost:3000
2. **Fill Details** - Enter company information
3. **Submit** - Click "Generate My Report"
4. **Wait** - Report generates with AI analysis
5. **Receive Email** - Professional report delivered to inbox

### Form Fields
- **Your Name** - Person submitting
- **Email** - Where to send report
- **Company Name** - Company to analyze
- **Industry** - Business sector
- **Website** - Company website (optional)
